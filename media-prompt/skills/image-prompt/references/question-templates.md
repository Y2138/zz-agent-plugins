# Question Templates

Load this file only when you need concrete wording for follow-up questions.

## First Reply For Very Short Prompts

```text
可以，我先帮你把这个画面补全成更好出的版本。先确认两个最影响结果的点：
1. 这张图是打算做壁纸、海报、社媒配图，还是普通插画？
2. 你想要更偏写实、电影感、动漫感，还是绘画感？
```

Compressed version:

```text
可以。先告诉我两点就行：用途是什么，以及你想要写实、电影感还是插画感？
```

## Reusable Questions

Use case:

```text
这张图主要拿来做什么？我会按用途决定构图和比例。
可选方向：壁纸、海报、社媒封面、文章配图、故事插画。
```

Style:

```text
你更想要哪种视觉方向？
可选方向：写实摄影、电影感、日漫/二次元、油画、水彩、3D 渲染、极简插画。
```

Scene:

```text
这个“日出”你希望出现在什么场景里？
可选方向：海边、山间湖泊、城市天际线、乡野田野、森林、沙漠。
```

Composition:

```text
画面更想要哪种构图？
可选方向：大场景远景、有人物或主体的中景、聚焦细节的近景。
```

Light and mood:

```text
氛围你更偏哪种？
可选方向：温暖治愈、清冷安静、史诗震撼、梦幻朦胧、极简空灵。
```

Constraints:

```text
有没有必须加上或必须避开的元素？
比如：不要人物、不要文字、要有云海、要有小船、画面尽量干净。
```

## "You Decide" Fallback

```text
那我按更通用、出图稳定的方向来：做成电影感写实风，大场景远景，暖金色晨光，画面干净，适合 16:9 横图。如果你想改成海边、城市或插画风，我也可以马上给你换一版。
```
